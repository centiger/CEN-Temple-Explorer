## 이번 반영
- 헤롯 성전 Explorer 활성화
- `assets/herod/scenes/` 13개 장면 추가
- `assets/herod/thumbs/` 썸네일 추가
- `data/herod.js` 탐험 순서 구성
- 공통 Explorer Engine은 수정하지 않음
- Service Worker 캐시명 갱신
